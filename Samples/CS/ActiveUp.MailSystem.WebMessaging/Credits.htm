Credits :
ActiveWebMessaging has been developed by ActiveUp SPRL, a Belgian .NET development company.
ActiveWebMessaging is powered by the following systems :
  •   Microsoft ASP.NET :

    All pages are ASP.NET pages and the underlying software is written in C#, a .NET language.

  •   ActiveUp's ActiveMail :

    Communications with the underlying server (parse, manipulate messages, folders, send messages) are made using ActiveMail (IMAP4 and SMTP).

  •   ActiveUp's HtmlTextBox :

    ActiveUp's HtmlTextBox is used as online editor to type the messages.

  •   Cascading Style Sheets :

      ActiveWebMessaging uses valid CSS2 style sheets.
